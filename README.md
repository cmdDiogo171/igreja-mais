# ⛪ Igreja Mais

Site institucional desenvolvido para facilitar o acesso às informações da Igreja Mais, conectando membros e visitantes de forma simples, rápida e acessível.

## 🌐 Sobre o Projeto

O site reúne as principais informações da igreja em um único lugar, permitindo que qualquer pessoa encontre facilmente horários de cultos, localização, informações sobre células e canais de comunicação.

## ✨ Funcionalidades

- 📍 Localização da igreja integrada ao Google Maps
- 📅 Horários de cultos e reuniões
- 👥 Página de células
- 📖 Informações sobre ministérios
- 📱 Acesso rápido ao WhatsApp
- 👍 Integração com Facebook
- 📞 Informações de contato
- 💻 Layout responsivo para celulares, tablets e computadores

## 🎯 Objetivo

Proporcionar uma presença digital moderna para a Igreja Mais, facilitando a comunicação com membros, visitantes e pessoas que desejam conhecer a igreja.

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Git
- GitHub

## 📱 Responsividade

O projeto foi desenvolvido para oferecer uma experiência agradável em diferentes dispositivos:

- Desktop
- Tablet
- Smartphone

## 📸 Principais Seções

### 🏠 Página Inicial
Apresentação da igreja e acesso rápido às principais informações.

### ⛪ Horários de Culto
Informações atualizadas sobre os cultos e reuniões semanais.

### 👥 Células
Lista de células com informações para participação e contato.

### 📍 Localização
Mapa integrado para facilitar o acesso à igreja.

### 📞 Contato
Links diretos para WhatsApp, Facebook e demais canais oficiais.

## 🚀 Acesso

O projeto pode ser acessado através do GitHub Pages.
